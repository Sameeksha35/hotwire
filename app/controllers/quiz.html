Rails Practice Quiz

1. What command generates a new Rails application?

Answer: rails new app_name

2. What file is used to configure database settings in a Rails application?

Answer: config/database.yml

3. How do you start the Rails server?

Answer: rails server or rails s

4. Which command is used to generate a new model?

Answer: rails generate model ModelName

5. How do you create and apply migrations?

Answer: rails db:migrate after generating migrations.
